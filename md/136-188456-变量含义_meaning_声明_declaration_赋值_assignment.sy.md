### 赋值

- 其中等于号`=`是一个`赋值`运算符
	- 赋值就是把一个值给一个变量
	- 他的意思是把右边的数值给到左边去
	- 就是 ←
- x = 0
  - 就是 x ← 0
  - 将 0 这个数值
	- 放到 x 这个容器里面去
  - 也叫做将 0 赋给 x
 
![图片描述](https://doc.shiyanlou.com/courses/uid1190679-20220519-1652964309290)

- 反过来行么？

### 反过来

- 0 = x
  - 不成立
  - 因为 0 ← x
  - 0 是 数值
  - x 是 容器
  - 容器 里面的东西
	- 没法放到一个具体的数值上去
		- cannot assign to literal

![图片描述](https://doc.shiyanlou.com/courses/uid1190679-20220519-1652964375007)

- assign 指的是指定分配
	- 这里用=表示进行分配
	- o = 0 就是
		- 把 0这个值 赋给 叫o的这个变量名
- 怎么赋值的呢？

### 赋值过程

- 左边是标识符
  - 他可以标识内存中的一个容器空间
- 右边是具体的值
  - 容器空间里的东西可以变
- 就像往杯子里倒不同的液体
	- 茶
	- 咖啡
	- 牛奶
